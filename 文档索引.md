## 快速参考

### 常用API端点
```
基础URL: http://localhost:3000

产品管理:
GET    /api/products              # 获取产品列表
POST   /api/products              # 创建产品
PUT    /api/products/{name}       # 重命名产品
DELETE /api/products/{name}       # 删除产品

文件操作:
GET    /api/folder/{path}/details # 获取文件夹详情
POST   /api/upload-files          # 上传文件
POST   /api/delete-file           # 删除文件

国际化:
GET    /api/i18n/translations     # 获取翻译
POST   /api/i18n/translations     # 更新翻译
```

### 常用命令
```bash
# 开发环境启动
npm run start

# 生产环境构建
npm run build
npm run prod

# 进程管理
pm2 start ecosystem.config.js
pm2 status
pm2 logs product-management
```

### 重要目录
```
/Product/          # 产品文件存储
/uploads/          # 上传文件存储
/src/components/   # 前端组件
/server/           # 后端服务代码
/public/data/      # 静态数据文件
```
